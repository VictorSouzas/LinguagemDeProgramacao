Exercício de ArrayList
●
Implementar uma solução usando OO para o problema a 
seguir:
–
Criar uma classe Contato para armazenar o nome, 
endereço, telefone e e-mail de seus contatos.
●
Utilizar o conceito de encapsulamento
●
Definir dois construtores
–
Um deles com parâmetros
–
O outro sem parâmetros (faça uma chamada ao 
construtor com parâmetros)
●
Definir um método para imprimir os dados do 
contato

Implementar uma classe de gestão que tenha um 
atributo List do tipo Contato, contendo os seguintes 
métodos:
–
Adicionar contato
–
Buscar contato pelo nome e retorna o objeto
–
Excluir todos os contato de um nome
–
Modificar o telefone de um contato pelo nome
–
Visualizar contatos 

●
Criar uma classe AppContato com as seguintes opções 
de menu:
1. Cadastrar Contato
2. Buscar Contato pelo nome e exibir seus dados
3. Modificar telefone do Contato
4. Excluir todos os contatos da agenda que possuam 
um nome informado pelo usuário. O programa deverá 
informar quantos registros foram removidos.
5. Visualizar todos Contatos
6. Sair

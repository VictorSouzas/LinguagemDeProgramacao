Exercício
●
Crie a classe Animal contendo nome, raça, cor e ano de 
nascimento.
–
Não esqueça de criar pelo menos um construtor.
–
Os atributos devem ser encapsulados.
–
Desenvolva uma operação para imprimir os dados do animal.
●
Crie uma classe de gerência de uma lista de animais que:
–
Adicione um animal.
–
Informe quantos animais existem de uma raça pesquisada pelo 
usuário.
–
Liste todos os animais cadastrados.
●
Implemente uma classe Principal que deverá permitir o usuário (1) 
cadastrar animais, (2) buscar quantos animais existem de uma 
raça, (3) apresentar todos e (4) sair.
